# CampFinder
CampFinder is a website that helps users find and share information about the best campgrounds around the world.

## Tools Used

This project was created using JavaScript, Node.js, Express, MongoDB, and Bootstrap. Geocoding was done using Mapbox, while the Passport.js was used to handle authentication.
